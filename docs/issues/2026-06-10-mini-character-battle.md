# 戦闘キャラクターをミニキャラ表現に寄せる

## メモ

戦闘画面のキャラクターは、等身高めの立ち絵ではなく、Library of Ruina のようなミニキャラ風の見た目で戦わせたい。

## 方向性

- 背景ステージ上に小さめのキャラを配置する
- キャラは輪郭が分かりやすいシルエットにする
- 待機・攻撃・被ダメージなどの戦闘アニメーションもミニキャラ前提で作る
- HUD用アイコンや会話用立ち絵とは別の戦闘用画像として管理する

## 攻撃時のカメラ演出

攻撃時は、ただ弾やエフェクトを飛ばすのではなく、自分のキャラクターを主役にした画面演出にしたい。

やりたい流れ:

1. 攻撃開始時に、自分のキャラクターへカメラを寄せてアップ表示する
2. 自分のキャラクターが敵へ近づく間、カメラもキャラを追従する
3. 敵の近くまで寄ったら攻撃モーションに入る
4. ヒット時に敵側へインパクトを出す
5. 攻撃後は元の引き画面へ戻る

実装で気をつけること:

- カメラ演出は Pixi の stage 全体、またはキャラクターと背景をまとめたコンテナの transform で扱う
- キャラクターの移動、カメラ追従、攻撃モーション、ヒット演出を同じタイムラインで管理する
- 攻撃中も UI やコードエディタは不自然に揺らさない
- 最初はプレイヤー攻撃だけでよい。敵攻撃や複数段攻撃への拡張は後で考える

受け入れ条件:

- プレイヤー攻撃時に、キャラクターがアップになってから敵へ近づく
- 移動中はカメラがキャラクターを追っているように見える
- 敵の近くで攻撃画像または攻撃モーションに切り替わる
- ヒット後、画面が元の構図へ自然に戻る

## 戦闘ヒット演出の素材準備メモ

まずは Pixi.js と anime.js で作れる汎用エフェクトを優先する。画像素材が必要なものは後回しにして、キャラ別エフェクトへ進む前に必要素材をそろえる。

### 画像素材なしで先に実装できる

- 着弾フラッシュ: Pixi の円形/矩形フラッシュで作る
- 円形衝撃波: Pixi Graphics のリングを anime.js で拡大・フェードさせる
- 画面揺れ: 既存の Pixi カメラ/コンテナ transform で揺らす
- ヒットストップ: 攻撃タイムライン上で短く待機する
- 敵の点滅: tint、alpha、blendMode で表現する
- 敵のノックバック: 敵スプライトの x/y を anime.js で戻す
- ダメージ数字: Pixi Text または HTML overlay で表示する
- 文字が砕ける演出: Pixi Text や小さな文字片を分割して飛ばす
- 透明な波紋: Pixi Graphics の薄いリングで作る
- 斬撃ライン: Pixi Graphics の細い線/三角形を高速に出す
- 画面暗転フラッシュ: Pixi 全画面矩形または CSS overlay で作る
- 残像: キャラスプライトを複製して alpha/tint を下げる

### 素材なしでも仮実装できるが、見た目を上げるなら素材があるとよい

- 紙片・文字片の飛散: 仮は Pixi Graphics と文字で作れる。質感を出すなら紙片PNG、文字片テクスチャ、手書き風フォントがあるとよい
- 黒い塵・灰: 仮は黒い小円パーティクルで作れる。質感を出すなら小さい塵/灰テクスチャがあるとよい
- 斬撃ライン: 仮は線で作れる。キャラ別にするなら筆跡・光跡の透過PNGがあるとよい
- ダメージ数字: 仮は通常フォントで作れる。作品らしさを出すなら数字用フォント、または数字スプライトがあるとよい

### 先に素材を用意したほうがよい

- 魔法陣が崩れる演出: 魔法陣のデザインが絵柄に直結するため、SVG、透過PNG、または Lottie がほしい
- 地面のひび割れ: 背景とのなじみが重要なので、ひび割れの透過PNG/SVG、または Lottie がほしい

### 素材を用意する場合の目安

- 形式: 透過PNGかSVG。動きが決まっているものだけ Lottie
- 色: 白/黒単色でもよい。実装側で tint できるようにする
- サイズ: 512px 四方以内を基本にする。小さい粒子素材は 64px 四方以内
- 命名: `src/assets/effects/common/` に `magic_circle.svg`, `crack_decal.png`, `paper_fragments.png` のように用途名で置く

### 初期実装の優先順

1. 着弾フラッシュ、円形衝撃波、敵点滅、ノックバック、ヒットストップ
2. ダメージ数字、画面揺れ、透明な波紋、紙片・文字片の飛散
3. 斬撃ライン、残像、黒い塵・灰
4. 魔法陣崩壊、地面ひび割れ
