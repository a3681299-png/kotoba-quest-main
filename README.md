# Kotoba Quest

プログラミングを学びながら楽しめる、コードバトルRPGゲームです。
日本語風の簡易プログラミング言語でコードを書いて魔法を唱え、敵を倒していきます。

## プロジェクト概要

「Kotoba Quest」は、子供でも楽しめるプログラミング学習ゲームです。
プレイヤーは日本語風のコマンド（`攻撃()`, `防御()`, `繰り返す()` など）を使ってコードを書き、
バトルで敵を倒しながらプログラミングの基礎概念を学びます。

Peggyパーサージェネレーターで独自言語を実装し、Pixi.jsでリアルタイムアニメーション、
Reactでインタラクティブなエディタを提供しています。

## 実装済みの機能

### ✅ ゲームシステム
- **6つのステージ**: 段階的に難易度が上がる学習体験
  - ステージ1: はじめての魔法（攻撃の基本）
  - ステージ2: 連続攻撃をマスター（ループ）
  - ステージ3: 防御の重要性（防御コマンド）
  - ステージ4: 変数でパワーアップ（変数宣言）
  - ステージ5: 状況を見て判断（条件分岐）
  - ステージ6: ドラゴン討伐【ボス】（総合課題）

- **インタラクティブチュートリアル**: 初心者向けの段階的ガイダンス機能
- **ステージクリアシステム**: 次のステージへの進行とリトライ機能

### ✅ プログラミング言語機能
- **基本コマンド**: `攻撃()`, `防御()`, `回復()`
- **ループ構文**: `繰り返す(回数) { ... }`
- **条件分岐**: `もし(条件) { ... }`
- **変数システム**: `変数 名前 = 値` で変数宣言・参照
- **式評価**: 比較演算子（`<`, `>`, `<=`, `>=`, `==`, `!=`）

### ✅ ゲームエンジン
- **Pixi.js バトルシーン**: スムーズな戦闘アニメーション
  - 攻撃エフェクト（ファイア、アイス、サンダー）
  - ダメージ表示とHPバー更新
  - 敵の外観変化（ダメージ受信時の視覚フィードバック）
  - 勝利・敗北アニメーション

- **敵AIシステム** (`EnemyAI.ts`):
  - 敵毎の行動パターン（攻撃、防御、回復）
  - 敵の意図（Intent）表示システム
  - ダメージ計算とバランス調整

- **スペル実行エンジン** (`SpellExecutor.ts`):
  - AST（抽象構文木）の解釈実行
  - イベント駆動型の非同期実行
  - ステップ実行モード（デバッグ用）

### ✅ UI/UXコンポーネント
- **コードエディタ** (`CodeEditor.tsx`):
  - シンタックスハイライト風の表示
  - 実行中の行のハイライト
  - エラー表示とフレンドリーなエラーメッセージ

- **デバッグパネル** (`DebugPanel.tsx`):
  - ステップ実行機能
  - 実行ログとアクション履歴
  - 変数の状態表示

- **インテント表示** (`IntentDisplay.tsx`):
  - 敵の次の行動の可視化

- **バトル画面** (`BattleScreen.tsx`):
  - HP表示、ターンカウント
  - ステージ情報とヒント
  - クリア・失敗時のダイアログ

### ✅ 状態管理
- **Zustand ストア** (`useGameStore.ts`):
  - プレイヤー/敵のHP管理
  - バトルフェーズ管理（プレイヤーターン/敵ターン/勝利/敗北）
  - 変数スコープの管理
  - リトライ用のスナップショット機能
  - ステップ実行のための一時停止/再開機能

### ✅ テスト
- **パーサーのユニットテスト** (`parser.test.ts`):
  - 文法の正確性を検証
  - 攻撃、防御、ループ、条件分岐、変数宣言のテスト

## 技術スタック

- **Language**: TypeScript
- **Framework**: React 19
- **Build Tool**: Vite
- **Rendering**: Pixi.js v8
- **Parser Generator**: Peggy
- **State Management**: Zustand
- **Testing**: Vitest

## ゲームの遊び方

1. ブラウザで開発サーバーにアクセス
2. コードエディタに日本語風のコマンドを入力
3. 「▶️実行」ボタンでコードを実行
4. バトルアニメーションを見ながら敵を倒す
5. ステージクリアで次のステージへ進行

### サンプルコード例

**ステージ1: 攻撃の基本**
```
攻撃("ファイア")
```

**ステージ2: ループ**
```
繰り返す(3) {
  攻撃("ファイア")
}
```

**ステージ4: 変数**
```
変数 威力 = 25
攻撃("ファイア")
防御()
```
